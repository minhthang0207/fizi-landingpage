- hooks: useMediaQuery: 
Hook này giúp component React "lắng nghe" sự thay đổi của kích thước màn hình hoặc các điều kiện hệ thống khác. Khi điều kiện thay đổi (ví dụ: người dùng xoay điện thoại hoặc co giãn trình duyệt), hook sẽ kích hoạt re-render để giao diện cập nhật tương ứng.

useMediaQuery là một Custom Hook. Nhiệm vụ của nó là: Theo dõi kích thước (hoặc trạng thái) màn hình và báo về cho React biết để thay đổi giao diện tương ứng.

Nó giống như việc bạn viết CSS Media Queries như @media (max-width: 768px), nhưng thay vì viết trong file CSS, bạn đưa nó vào trong file JavaScript/TypeScript để điều khiển logic của component.
